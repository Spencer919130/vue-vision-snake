## Design Concept
「Vision Snake」旨在打破傳統貪食蛇僵硬的按鍵操作，結合 MediaPipe Hands 打造更直覺的空間肢體互動。核心理念在於「化繁為簡」與「風險控制」的結合。傳統 AI 絕對座標追蹤常因人手生理抖動與鏡頭雜訊，讓玩家產生嚴重的挫折感。因此，本專案創新設計了「十字滑軌導航機制」，強迫食指尖的定位點死死吸附於畫布中央的雙軸虛擬軌道上，讓隔空操控擁有如實體機台搖桿般的扎實度與高容錯率。

此外，遊戲更引入了極具策略性的「進退檔暴走機制」：玩家平日能以食指優雅導航，一旦遇到緊急狀況或想挑戰高分時，只需用力握拳，便會瞬間點火進入極速衝刺狀態。在此狀態下得分大幅翻倍，但同時蛇身會無條件被迫劇烈增長。這逼使玩家必須在「高分貪婪」與「生存壓力」之間做出即時的肉體反應與節奏判斷。玩家在遊玩後，將體驗到從「按鍵操作」轉向「身體流」的強烈節奏差異，這不僅是一場遊戲，更是一場關於身體協調與反應速度的極限挑戰。

## How My Version Is Different
我的版本和原版 Snake 最大差異是什麼？

空間維度的交互進化：完全捨棄傳統的鍵盤 W/A/S/D 或方向鍵，改用 MediaPipe Hands 實時捕捉「食指指尖」在三維空間映射出的相對象限。

滑軌物理吸附機制：畫面上的青色定位核心被限制只能在背景的 X 軸與 Y 軸上滑行（如同磁鐵吸附在滑軌上），完美過濾了手部顫抖的雜訊，讓轉向判斷極度精準、簡單。

主動式風險懲罰（暴走機制）：原版只能被動避障，本專案加入「握拳暴走」。玩家可主動握拳踩下油門，在享受速度與雙倍分數的同時，必須承擔「身體每 0.6 秒無條件自動變長」的自殺風險，將策略性與刺激感拉滿。

## How To Play
玩家要怎麼操作？

進入艙門：打開網頁並允許瀏覽器存取攝影機權限，將單手放入鏡頭預覽畫面中。

食指軌道導航：伸出食指（維持 1~5 根手指張開狀態）作為虛擬搖桿：

將食指移到畫面上方 / 下方（中央縱軸軌道），蛇便會向上 / 向下轉向。

將食指移到畫面左方 / 右方（中央橫軸軌道），蛇便會向左 / 向右轉向。

👊 握拳點火暴走：在鏡頭前用力握緊拳頭（偵測到 0 根手指），畫面會切換為紅色霓虹色調並引發微幅震動，蛇會立刻以兩倍速狂飆，吃到食物分數大幅加成，但長度會隨時間自動暴增。手掌再度張開即可退檔減速。

敗北條件：蛇頭撞擊到四週的青色（正常）或紅色（暴走）霓虹外框，或者蛇頭撞擊到自己的身體，即 Game Over，可點擊「再次挑戰」重新開車。
